# Qualitative Artifacts

- `qualitative_selection.csv` records the deterministic selection of one representative Road200 case per scenario.
- `qualitative_small_distant_profiles.png` is the supplementary small/distant output-profile example referenced by the paper.
- The crowded/occluded case used in the main paper is available under `paper/figures/`.

The CSV uses repository-independent relative image paths. Raw BDD100K images and original annotations are not redistributed in this repository.
